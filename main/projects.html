Dialog ESMS API

It is an API package to handle a dialog ESMS service build using javascript. I published it an npm package library and you can get deatails here (@eshanchathuranga/esms-api)

How it work

Frist you want to Dialod ESMS gateway account. Contact Dialog Axiata PLC to get deatails.
